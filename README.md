# 🐱 English for Nana

Простое и удобное веб-приложение для изучения английского языка, созданное специально для мамы. 
Приложение помогает учить базовые фразы, числа и повседневные выражения с помощью карточек и озвучки.

## ✨ Основные функции

* **Два режима работы:**
    * 📖 **УЧИТЬ:** Режим карточек, где сразу видны перевод, транскрипция и кнопка озвучки.
    * 🧠 **ПРОВЕРКА:** Режим тестирования. Видна только русская фраза, английский вариант скрыт до нажатия кнопки.
* **Темы на выбор:** Удобное меню для переключения между темами (Приветствия, Кухня, Здоровье, Сон, Числа).
* **Тренажер чисел:** Полный список чисел от 1 до 20 для ежедневной практики.
* **Система очков:** В режиме проверки приложение считает правильные и неверные ответы.
* **Голосовое сопровождение:** Встроенная озвучка (Text-to-Speech) для тренировки правильного произношения.

## 🚀 Как запустить

Приложение работает прямо в браузере и не требует установки. 

1. Перейдите по ссылке: `https://product-girl.github.io/English-for-Nana/`
2. **Для iPad/iPhone:** Нажмите кнопку «Поделиться» в Safari и выберите **«На экран „Домой“»**, чтобы использовать приложение как обычную иконку на рабочем столе.

## 🛠 Технологии

* **HTML5 / CSS3** (адаптивная верстка для планшетов).
* **JavaScript** (логика переключения режимов и фильтрация тем).
* **Web Speech API** (для синтеза речи).
